\# Module 3 — HTML Headings



HTML provides six heading levels used to structure content on a webpage.



\## Heading Tags



HTML headings range from `<h1>` to `<h6>`.



\* `<h1>` → main heading

\* `<h2>` → section heading

\* `<h3>` → subsection

\* `<h4>`

\* `<h5>`

\* `<h6>` → smallest heading



Search engines and browsers use headings to understand the structure of a page.



\## Example



```html

<h1>Main Heading</h1>

<h2>Section Heading</h2>

<h3>Subsection Heading</h3>

<h4>Sub-level Heading</h4>

<h5>Minor Heading</h5>

<h6>Smallest Heading</h6>

```
