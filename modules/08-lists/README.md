\# Module 8 — HTML Lists



Lists are used to group related items.



HTML supports two main types of lists:



\## Unordered List



Uses `<ul>` (bullets)



```html

<ul>

<li>Apple</li>

<li>Banana</li>

<li>Orange</li>

</ul>

```



\## Ordered List



Uses `<ol>` (numbers)



```html

<ol>

<li>Step 1</li>

<li>Step 2</li>

<li>Step 3</li>

</ol>

```



\## Nested Lists



Lists can be placed inside other lists.



```html

<ul>

<li>Fruits

&nbsp;  <ul>

&nbsp;     <li>Apple</li>

&nbsp;     <li>Orange</li>

&nbsp;  </ul>

</li>

</ul>

```
