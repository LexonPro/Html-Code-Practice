\# Module 13 — HTML Div and Span



`<div>` and `<span>` are container elements used to group HTML content.



\## `<div>` Element



\* Block-level container

\* Used to group sections of a webpage

\* Often used for layout structure



Example:



```html

<div>

<h2>Section Title</h2>

<p>This is a section of content.</p>

</div>

```



\## `<span>` Element



\* Inline container

\* Used to style or highlight small portions of text



Example:



```html

<p>This is <span>highlighted text</span> inside a paragraph.</p>

```
