\# Module 4 — HTML Paragraphs



Paragraphs are used to display blocks of text in HTML.



The `<p>` tag defines a paragraph.



Browsers automatically add space before and after paragraphs.



\## Line Breaks



To insert a line break inside text, HTML uses the `<br>` tag.



\## Horizontal Rule



The `<hr>` tag creates a horizontal line used to separate sections.



\## Example



```html id="ozsnra"

<p>This is a paragraph.</p>



<p>This is another paragraph.</p>



Line one<br>

Line two<br>



<hr>

```
