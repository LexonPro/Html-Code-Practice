\# Module 5 — HTML Text Formatting



HTML provides several tags to format text and emphasize content.



\## Common Formatting Tags



\* `<b>` → Bold text

\* `<strong>` → Important text

\* `<i>` → Italic text

\* `<em>` → Emphasized text

\* `<mark>` → Highlighted text

\* `<small>` → Smaller text

\* `<del>` → Deleted text

\* `<ins>` → Inserted text

\* `<sub>` → Subscript text

\* `<sup>` → Superscript text



These tags help structure text and improve readability.



\## Example



```html

<p><b>Bold text</b></p>

<p><strong>Important text</strong></p>

<p><i>Italic text</i></p>

<p><em>Emphasized text</em></p>

<p><mark>Highlighted text</mark></p>

<p><small>Smaller text</small></p>

<p><del>Deleted text</del></p>

<p><ins>Inserted text</ins></p>

<p>H<sub>2</sub>O</p>

<p>2<sup>2</sup> = 4</p>

```
