\# Module 17 — HTML Meta Tags



Meta tags provide information about a webpage that is not visible to users.



They are placed inside the `<head>` section.



\## Common Meta Tags



\### Character Encoding



```html id="z1k9r3"

<meta charset="UTF-8">

```



Defines character encoding.



\### Viewport (Responsive Design)



```html id="p6t2m8"

<meta name="viewport" content="width=device-width, initial-scale=1.0">

```



Makes the page responsive on mobile devices.



\### Description (SEO)



```html id="q4v7c1"

<meta name="description" content="HTML learning modules">

```



Provides a description for search engines.



\## Example



```html id="w8y5s2"

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<meta name="description" content="HTML tutorial">

<title>Meta Tags Example</title>

</head>

```
