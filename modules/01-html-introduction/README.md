\# Module 1 — HTML Introduction



\## What is HTML?



HTML (HyperText Markup Language) is the standard language used to create and structure web pages.



It defines the structure of web content using \*\*elements and tags\*\*.



---



\## How HTML Works



1\. You write HTML code.

2\. A web browser reads the code.

3\. The browser renders the page visually.
