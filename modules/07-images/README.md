\# Module 7 — HTML Images



Images are added to web pages using the `<img>` tag.



\## Basic Syntax



```html

<img src="image.jpg" alt="description">

```



\## Important Attributes



\* `src` → image file path

\* `alt` → alternative text if the image cannot load

\* `width` → image width

\* `height` → image height



The `<img>` tag does not have a closing tag.



\## Example



```html

<img src="example.jpg" alt="Sample Image" width="300">

```
