\# Module 12 — HTML Buttons



Buttons allow users to perform actions such as submitting forms or triggering scripts.



\## Button Syntax



```html id="p4y7e2"

<button>Click Me</button>

```



\## Button Types



\* `submit` → submits form data

\* `reset` → clears form fields

\* `button` → generic clickable button



\## Example



```html id="k1r5s9"

<button type="button">Click</button>

<button type="submit">Submit</button>

<button type="reset">Reset</button>

```
