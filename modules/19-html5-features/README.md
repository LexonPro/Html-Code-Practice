\# Module 19 — HTML5 Features



HTML5 introduced many new features to improve web development.



\## Key Features



\### 1. Semantic Tags



HTML5 provides tags like:



\* `<header>`

\* `<nav>`

\* `<section>`

\* `<article>`

\* `<footer>`



These improve structure and readability.



\### 2. Multimedia Support



HTML5 allows embedding media without plugins:



\* `<audio>`

\* `<video>`



\### 3. New Input Types



HTML5 forms include new input types:



\* `email`

\* `date`

\* `number`

\* `range`

\* `color`



\### 4. Canvas



HTML5 introduces `<canvas>` for drawing graphics using JavaScript.



\### 5. Local Storage



Web browsers can store data locally using:



\* `localStorage`

\* `sessionStorage`
