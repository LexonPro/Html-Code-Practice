\# Module 9 — HTML Tables



HTML tables are used to display tabular data.



\## Basic Table Tags



\* `<table>` → defines the table

\* `<tr>` → table row

\* `<th>` → table header

\* `<td>` → table data cell



\## Basic Table Structure



```html

<table>

<tr>

<th>Name</th>

<th>Age</th>

</tr>



<tr>

<td>John</td>

<td>25</td>

</tr>

</table>

```



\## Table Border Example



```html

<table border="1">

<tr>

<th>Course</th>

<th>Duration</th>

</tr>



<tr>

<td>HTML</td>

<td>2 Weeks</td>

</tr>

</table>

```
