\# Module 18 — HTML Entities



HTML entities are used to display reserved characters and symbols.



\## Why Use Entities?



Some characters have special meaning in HTML:



\* `<` and `>` are used for tags

\* `\&` is used for entities



To display them, we use HTML entities.



\## Common Entities



\* `\&lt;` → <

\* `\&gt;` → >

\* `\&amp;` → \&

\* `\&copy;` → ©

\* `\&nbsp;` → non-breaking space



\## Example



```html id="z8v2k5"

<p>5 \&lt; 10</p>

<p>AT\&amp;T</p>

<p>\&copy; 2026</p>

```
