\# Module 10 — HTML Forms



Forms collect user input on web pages.



\## Basic Form Structure



```html id="u4n7s2"

<form>

<label>Name:</label>

<input type="text">

</form>

```



\## Common Form Elements



\* `<form>` → defines a form

\* `<label>` → text label for input

\* `<input>` → user input field

\* `<textarea>` → multi-line text input

\* `<select>` → dropdown menu

\* `<option>` → items inside dropdown

\* `<button>` → clickable button



\## Example



```html id="r9p3k6"

<form>

<label>Name:</label>

<input type="text">



<label>Email:</label>

<input type="email">



<button type="submit">Submit</button>

</form>

```
