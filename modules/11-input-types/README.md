\# Module 11 — HTML Input Types



HTML forms use the `<input>` tag to collect user data.



Different types of inputs allow different kinds of data.



\## Common Input Types



\* `text` → single-line text

\* `password` → hidden characters

\* `email` → email address

\* `number` → numeric input

\* `radio` → select one option

\* `checkbox` → select multiple options

\* `date` → date picker

\* `file` → upload file

\* `submit` → submit form



\## Example



```html id="7d9f0a"

<input type="text">

<input type="password">

<input type="email">

<input type="number">

<input type="date">

<input type="file">

```
