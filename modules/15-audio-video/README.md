\# Module 15 — HTML Audio and Video



HTML5 provides built-in elements to play audio and video files in web pages.



\## Audio Element



```html id="s8r1t4"

<audio controls>

<source src="audio.mp3" type="audio/mpeg">

</audio>

```



Attributes:



\* `controls` → shows play/pause controls

\* `autoplay` → automatically plays media

\* `loop` → repeats the media



\## Video Element



```html id="p6y9d3"

<video width="320" controls>

<source src="video.mp4" type="video/mp4">

</video>

```



Attributes:



\* `width` and `height`

\* `controls`

\* `autoplay`

\* `loop`
