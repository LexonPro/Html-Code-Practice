\# Module 14 — HTML Semantic Tags



Semantic tags describe the purpose of different sections of a webpage.



They improve readability, accessibility, and SEO.



\## Common Semantic Tags



\* `<header>` → top section of the page

\* `<nav>` → navigation menu

\* `<section>` → section of content

\* `<article>` → independent content

\* `<aside>` → side content

\* `<footer>` → bottom section



\## Example Layout



```html id="u4f7a2"

<header>

<h1>Website Title</h1>

</header>



<nav>

<a href="#">Home</a>

<a href="#">About</a>

</nav>



<section>

<h2>Main Content</h2>

<p>This is a section of the webpage.</p>

</section>



<footer>

<p>Copyright 2026</p>

</footer>

```
