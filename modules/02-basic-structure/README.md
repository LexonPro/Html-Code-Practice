\# Module 2 — Basic HTML Structure



This module explains the fundamental structure of an HTML document.



\## Standard HTML Template



Every HTML page follows this structure:



\* `<!DOCTYPE html>` → defines HTML5

\* `<html>` → root element

\* `<head>` → metadata and title

\* `<body>` → visible content



\## HTML Structure Diagram



```

<!DOCTYPE html>

<html>

&nbsp;  <head>

&nbsp;     <title>Page Title</title>

&nbsp;  </head>



&nbsp;  <body>

&nbsp;     Content visible in browser

&nbsp;  </body>

</html>

```



\## Learning Goal



Understand how browsers interpret HTML documents.
