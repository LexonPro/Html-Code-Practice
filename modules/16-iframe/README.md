\# Module 16 — HTML iframe



The `<iframe>` element allows you to embed another webpage inside the current page.



\## Basic Syntax



```html id="f3k9r1"

<iframe src="https://example.com"></iframe>

```



\## Common Attributes



\* `src` → URL of the page to embed

\* `width` → frame width

\* `height` → frame height

\* `title` → description for accessibility



\## Example



```html id="v6s2d8"

<iframe src="https://www.wikipedia.org"

width="600"

height="400">

</iframe>

```



Iframes are commonly used to embed maps, videos, and external pages.
